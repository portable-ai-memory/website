title: Validator
description: Valide arquivos PAM de memory store, conversas e embeddings contra os JSON Schemas oficiais
---

:::caution[Em Breve]
O validator PAM online está em desenvolvimento.
:::

O PAM Validator permitirá que você cole ou envie um arquivo PAM JSON e o valide diretamente no navegador contra o schema. Ele fornecerá relatórios de erro detalhados mostrando exatamente quais campos falharam na validação e por quê.

Enquanto isso, você pode validar localmente usando `ajv-cli`. Consulte o [Guia de Validação](/pt-br/tools/validation-guide/) para instruções.
